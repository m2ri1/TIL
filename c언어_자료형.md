# C언어의 자료형이란?
* 변수의 종류를 의미
* 변수에 값을 담기 이전에 정수, 실수, 문자 등 어떤 값을 지닐지 자료형을 통해 정해주어야 함

***

## C언어 자료형 정리

*정수형*

<code>int</code> 64bit   
<code>short</code> 32bit   
<code>long</code>  32bit  
<code>byte</code> 16bit   

*실수형*   

<code>float</code>   64bit    
<code>double</code> 1byte

*유니코드 문자*   

<code>char</code> 16bit
