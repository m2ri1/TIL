## Json Web Token

JWT란?

- JSON 방식으로 정보를 전달하는 토큰
- 웹서비스의 로그인 인증 시스템에서 주로 사용된다
- 암호화된 문자열의 형태이다

### JWT의 구조

`Header 헤더`  
토큰의 타입과 사용하는 암호화 알고리즘 정보가 들어있다

`Payload 페이로드`  
실제로 전달할 내용이(예: 사용자 ID) 들어있다

`Signature 서명`  
헤더와 페이로드를 합친 다음, 키(secret key)를 사용해 생성한 암호화된 문자열이다. 이를 통해 JWT가 변조되지 않았는지 확인할 수 있다.
